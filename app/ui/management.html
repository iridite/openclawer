更新说明

OpenClaw 核心更新

OpenClaw 核心通过 npm 包管理器安装，支持在线更新：

  1. 点击上方“检查更新”按钮查看是否有新版本
  2. 如有新版本，点击"更新版本"按钮
  3. 系统将自动执行 npm install openclaw@latest
  4. 更新完成后需要重启 Gateway 使新版本生效

FPK 包更新

如需更新整个 oc-deploy 应用（包括管理界面、脚本等）：

  1. 访问飞牛应用中心
  2. 检查 oc-deploy 应用是否有更新
  3. 下载并安装最新版本的 FPK
  4. 应用会自动重启并应用新版本

注意事项：

  • 更新前建议备份配置文件（系统会自动备份）
  • 更新过程中 Gateway 会短暂中断服务
  • 更新后请检查配置文件是否需要调整
  • 如遇问题可查看日志：/var/apps/oc-deploy/var/openclaw.log

日志文件路径

以下是系统各类日志文件的存储位置，您可以直接复制路径查看详细日志：

应用日志

  • OpenClaw Gateway 日志：
    /var/apps/oc-deploy/var/openclaw.log
  • Management API 日志：
    /var/apps/oc-deploy/var/info.log

配置文件

  • OpenClaw 配置文件：
    /root/.openclaw/openclaw.json
  • 配置备份文件：
    /root/.openclaw/openclaw.json.backup.*

进程信息

  • Dashboard PID 文件：
    /var/apps/oc-deploy/var/app.pid
  • Gateway PID 文件：
    /var/apps/oc-deploy/var/gateway.pid

提示：

  • 您可以使用 SSH 连接到 NAS 后使用 tail -f 命令实时查看日志
  • 例如：tail -f /var/apps/oc-deploy/var/openclaw.log
  • 使用 Ctrl+C 退出实时查看模式
