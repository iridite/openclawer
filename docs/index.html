为什么选择 OpenClawer？

⚡

一键部署

无需复杂配置，在 fnOS 应用中心直接安装，几分钟内即可启动你的 AI 代理。告别繁琐的命令行操作。

🎨

可视化管理

直观的 Web 界面，模型配置、渠道管理、状态监控一目了然。推荐配置和高级配置分离，新手友好。

📊

实时监控

实时显示 Gateway 状态、CPU、内存使用情况。日志路径展示，便于快速排障。

🔌

多渠道支持

支持 Telegram、Discord、飞书、QQ 等多个消息平台。内置配置示例，快速接入。

🛠️

插件管理

插件状态检测和一键安装。QQ 渠道插件支持自动检测和安装，无需手动操作。

💾

配置管理

支持配置导入/导出/复制/恢复。JSON 校验异常提示但不强制阻断，灵活又安全。

快速开始

1

下载安装

访问 Releases 页面，下载最新 FPK 包并上传到 fnOS 应用中心安装

2

配置模型

在"模型"页添加你的 AI 模型，填写 API Key 和基础配置即可

3

添加渠道

在"渠道"页配置消息平台，参考内置示例快速接入

4

启动使用

在"概览"页启动 Gateway，开始使用你的 AI 代理！

技术特性

🏗️ 架构设计

用户 → fnOS App Center → Management Console (18790) → /dashboard 代理 → OpenClaw Gateway (18789)

📦 FPK 封装

完整的 fnOS FPK 应用包，包含安装向导、升级回调、配置管理等完整生命周期支持

🔧 API 接口

提供完整的 RESTful API，支持系统状态、配置管理、网关控制、插件管理等操作
